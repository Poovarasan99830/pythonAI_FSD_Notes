🔡 ASCII Concept (Quick Note)
❓ What is ASCII?

ASCII assigns numbers to characters.

| Character | ASCII Value |
| --------- | ----------- |
| A         | 65          |
| a         | 97          |
| 0         | 48          |

print(ord('A'))   # 65
print(ord('a'))   # 97
print(chr(65))    # A


ASCII means:🟢 American Standard Code for Information Interchange
ASCII is a standard that assigns numeric values to characters so that computers can store, process, and communicate text.


Computers understand only numbers 🤖
So ASCII converts:

Letters
Numbers
Symbols
into numbers



| Type | Range  |
| ---- | ------ |
| A–Z  | 65–90  |
| a–z  | 97–122 |
| 0–9  | 48–57  |
