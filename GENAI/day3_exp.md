Input-na data (numbers) neuron-kulla varudhu
🔹 2. Inputs & Weights
Inputs = features (data)
Weights = importance of each input

👉 Core: Important data gets higher weight




Salary important → weight high
Age less important → weight low
👉 Thanglish:
Weight-na “evlo mukkiyam” nu decide pannradhu



Small adjustment value add pannuvom
Helps model learn better
👉 Thanglish:
Bias-na oru extra tuning value (fine adjust)



“produces output”
Final calculation pannitu result kudukkum
👉 Formula:
Output = (Input × Weight) + Bias
👉 Thanglish:
Final-ah oru result varum (decision / prediction)




# ------------------------------------------------------    
🔹 Simple Real-Life Example -Bias
# ------------------------------------------------------    

Loan approval system imagine pannunga:

Input: salary = 50k
Weight: salary importance = 0.8
Bias: +10

👉 Calculation:
(50 × 0.8) + 10 = 50

👉 Output:
Loan approve pannalama nu decision
